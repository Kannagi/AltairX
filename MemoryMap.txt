0000 0048 DST
0000 0050 CMD
0000 0051 CORE_SELECT
0000 0052 SIZE

CMD : 00 NOP
CMD : 01 RAM -> SPM-I
CMD : 02 RAM -> SPM-D
CMD : 04 RAM <- SPM-I
CMD : 05 RAM <- SPM-D
CMD : 06 flush cache
CMD : 07 Start
CMD : 08 Stop

0000 0100 - 0000 01FF WRITE control
0000 0200 - 0000 0300 READ (0 nop,1 start,2 stop ,4 stop MMU )
